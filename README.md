# Instagram_Engagement_Analysis
#  Project Overview
This project analyzes Instagram post performance using data analysis techniques. 
It focuses on understanding how different factors like hashtags, followers, and post type affect engagement.
# Objectives
- Analyze engagement (likes, comments, shares)
- Find correlation between followers and likes
- Test whether hashtags improve engagement
- Identify best-performing post types
# Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
# Dataset
The dataset contains:
- Followers
- Likes
- Comments
- Shares
- Hashtags
- Post Type
- Caption Length
# Key Analysis
- Measure of central tendency (mean)
- Standard deviation (dispersion)
- Correlation analysis
- Data visualization
- Hypothesis testing
# Results / Insights
- Reels have higher engagement than images
- Positive correlation between followers and likes
- Hashtags improve engagement
- Engagement data shows slight variation
# Hypothesis Testing
- H₀: Hashtags do not affect likes  
- H₁: Hashtags affect likes
